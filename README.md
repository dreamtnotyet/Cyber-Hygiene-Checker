# 🛡️ Cyber Hygiene Checker

> **Your Personal Security Assistant** - A beginner-friendly Python tool that checks your digital security posture and provides actionable recommendations.

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

## ✨ Features

- 🔑 **Password Health Check** - Strength analysis and breach detection
- 📧 **Email Exposure Check** - Find if your email was in data breaches
- 🌐 **Browser Security** - Adblockers, HTTPS, security extensions
- 💻 **Software Updates** - OS and app update status
- 🔐 **2FA Status** - Two-Factor Authentication usage
- 💾 **Backup Status** - Data backup habits
- 👤 **Social Media Privacy** - Privacy settings review
- 📊 **Beautiful Reports** - HTML reports with visual scores
- 📈 **Progress Tracking** - Track your security score over time

## 🚀 Quick Start

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Installation

```bash
# Clone the repository
git clone https://github.com/dreamtnotyet/cyber-hygiene-checker.git
cd cyber-hygiene-checker

# Create a virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the tool
python main.py
